# Cardamom

A Go web app for storing bookmarks

## Build

Get fileb0x
```
go get -u github.com/UnnoTed/fileb0x
```

Build the app

```
make build
```

## Run

```
cp config.example.toml config.toml
./cardamom
```

Visit http://localhost:4000/

and sign up a user at http://localhost:4000/user/signup

## Usage

```
./cardamom -help
```
